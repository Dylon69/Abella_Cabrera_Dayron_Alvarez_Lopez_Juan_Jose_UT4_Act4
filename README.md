# Sistema de Gestión de Pedidos

Proyecto en Java que simula la gestión de pedidos de un restaurante.

## Funcionalidades
- Creación de pedidos con ID automático
- Gestión de artículos
- Control de máximo 5 artículos por pedido
- Cálculo automático del total
- Gestión de estados mediante enum
- Validaciones de datos
- Documentación con JavaDoc

## Estructura
- Articulo.java
- Pedido.java
- EstadoPedido.java
- Main.java
